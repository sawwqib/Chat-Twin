# CHAT TWIN
## 🤖 Telegram Chat Style Cloning Bot

A Telegram bot that can mimic and replicate a person’s texting style using their chat history. Upload conversations, and the bot learns tone, wording, and emoji usage to respond like that person.

---

## 🚀 Features

* 📂 Upload chat history
* 🧠 Learns texting patterns (tone, slang, emojis)
* 💬 Generates human-like replies
* ⚡ Fast Telegram bot integration
* 🎭 Mimics personality realistically


---

## 🛠️ Tech Stack

* Python
* Telegram Bot API
* NLP / AI Model (OpenAI or custom model)
* JSON / Text processing

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/sawwqib/Chat-Twin.git
cd Chat-Twin
```

### 2. Install dependencies

```bash
pip install python-telegram-bot==21.6 aiohttp
```

### 3. API's

```bash
Replace Telegram Bot API & Pollinations API
```

### 4. Run the bot

```bash
python main.py
```

---

## 📂 How It Works

1. Upload chat history { WhatsApp (.txt) / Telegram (.json) / Instagram (.json) / Discord (.txt / .json) } export
2. Bot processes messages
3. Extracts:

   * Writing style
   * Emoji patterns
   * Common phrases
4. AI model generates responses based on learned behavior
